# 📘 FMS v2.0 사용자 가이드

이 문서는 Football Management System의 주요 기능과 사용 방법을 설명합니다.

## 1. 선수 명단 관리 (Player Management)
- **기능**: 선수 데이터를 추가, 수정, 삭제하고 Firebase 데이터베이스와 동기화합니다.
- **사용법**:
  1. 화면 중앙의 표(Data Editor)에서 선수 정보를 직접 수정합니다.
  2. 새로운 선수를 추가하려면 표의 마지막 행에 입력합니다.
  3. 수정이 완료되면 반드시 **`💾 변경사항 저장`** 버튼을 눌러야 반영됩니다.
  4. **`🔄 Firebase와 동기화`** 버튼으로 서버 데이터를 불러올 수 있습니다.
  5. **`👥 테스트 선수 생성`** 버튼은 개발용 더미 데이터를 생성합니다 (주의: 기존 데이터 덮어씀).

## 2. 선수 평가 (Player Evaluation)
- **기능**: 선수들의 누적 스탯과 경기 기록을 조회합니다.
- **사용법**:
  - 상단 표에서 전체 선수의 총 출전 수와 평균 평점을 확인합니다.
  - 하단 드롭다운 메뉴에서 특정 선수를 선택하면, 해당 선수의 과거 경기 기록(날짜, 상대, 평점 등)을 볼 수 있습니다.

## 3. AI 라인업 생성 (AI Lineup Generation)
- **기능**: 경기 전략을 설정하고 AI 알고리즘을 통해 최적의 쿼터별 라인업을 자동 생성합니다.
- **사용법**:
  1. **경기 설정**: 쿼터 수, 포메이션, 인당 최대/최소 쿼터, 연속 휴식 금지 여부를 설정합니다.
  2. **참석 인원**: 금일 경기에 참석한 인원을 체크합니다.
  3. **GK 설정**: 골키퍼를 볼 선수들을 선택하고, 각 선수의 **GK 쿼터**와 **필드 쿼터**를 각각 지정합니다. (합계가 전체 쿼터 수와 맞아야 함)
  4. **컨디션 관리**: 부상이나 컨디션 난조가 있는 선수를 선택하면 최대 2쿼터로 제한됩니다.
  5. **`🚀 AI 라인업 분석 및 생성`** 버튼을 클릭합니다.
  6. 생성된 결과(요약표, 쿼터별 경기장 이미지)를 확인합니다.
  7. 하단의 **`🗓️ 경기 일정 생성`** 폼에 날짜와 상대 팀명을 입력하고 저장하여 경기를 확정합니다.

## 4. 경기 목록 (Match List)
- **기능**: 생성된 경기 일정과 결과를 확인합니다.
- **표시 정보**:
  - 날짜, 상대 팀, 상태(예정/종료), 점수 결과.

## 5. 경기 결과 기록 (Match Result Recording)
- **기능**: 경기가 끝난 후 스코어와 선수 평점을 입력하여 데이터를 누적합니다.
- **사용법**:
  1. 드롭다운에서 '예정(Scheduled)' 상태인 경기를 선택합니다.
  2. **스코어 입력**: 홈/원정 점수를 입력합니다.
  3. **기록 입력**: 득점자와 도움 선수를 선택합니다.
  4. **평점 입력**: 출전한 선수들의 활약상을 1.0~10.0 사이의 슬라이더로 평가합니다.
  5. **`경기 종료`** 버튼을 누르면 경기가 '종료(Completed)' 상태로 변경되고, 선수들의 누적 평점과 출전 수가 자동 업데이트됩니다.

---
**Tip**: 모든 데이터는 `matches.json`과 `players.json`에 로컬 저장되며, Firebase가 연결된 경우 클라우드에도 자동 백업됩니다.
